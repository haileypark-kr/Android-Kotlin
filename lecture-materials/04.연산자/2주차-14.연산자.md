# 연산자 종류

1. 산술
    - +, -, *, /, %
2. 대입
    - = (할당)
    - `A = B`
    - `x = 2`
3. 복합대입
    - 산술 연산자 + 대입 연산자 한 번에 사용
    - +=, -=, *=, /=, %=
4. 증감
    - ++, -- (변수 앞, 뒤로)
    - `A--`, `B++`
    - `--A`, `++B`
5. 비교
    - > , >=, <, <=
    - ==, !=: 같은가, 다른가
    - ===, !==: 객체가 같은가, 객체가 다른가
    - 결과는 항상 boolean
6. 논리
    - &&, || , ! (참 거짓 바꾸는 연산자)
    - `!( A && B )`
